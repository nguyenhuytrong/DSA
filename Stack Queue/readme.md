### Queue
- FIFO: first in first out 
- Some common def `.push()`, `.pop()`, `.empty()`, `.front()`, `.back()`, `.size()`, 
- `q.front()`: the first element; `q.back()`: the last element
- `q.pop()`: remove the front element
- `q.push(x)`: add the element to the back
